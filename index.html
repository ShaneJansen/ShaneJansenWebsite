  • BulkTesterAndroid >> While being a teaching assistant for an Android class, it was a huge pain to open each student's project, run it, and check every edge case in the application. It took many hours to grade a single lab! To help with this issue, I wrote a simple python script that will automatically run each student's project and check all of the required cases. All you have to do is write a unit test with Espresso for the lab and specify resource IDs. I can now grade everyone's lab in significantly less time.

I now use this library in all of my Android projects. The library also makes it easier to use Fragments by automatically handling the backstack, toolbar, and interaction between the Fragment and Activity.
